## Windows 11/operating system

- How to extract text from images on Windows 11
	- `Win +Shift+T`
- Snipping tool
	- `Win + Shift + S`
- Search bar
	- `Win + S`
- Open task manager
	- `Ctrl + Shift + Esc`
- Change keyboard language
	- `Win + Space`
## Chrome

- Switch to URL bar to quickly type in
	- `Ctrl + L``
- Move tab left or right
	- `Ctrl + Shift + Pgup/Pgdn`
